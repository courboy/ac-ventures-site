The AI Engine

What each agent actually does.
From mundane to complex.

ANALYST

Deal Evaluation Engine

15-20 hrs/week
Mundane

Document Extraction

Parses IMs, rent rolls, leases, valuations → extracts key data points (rent/sqm, cap rate, lease terms, tenant info) → structures into database

Medium

Comparable Assembly

Queries RCA database for relevant transactions → filters by geography, asset type, size, timing → assembles comp tables with $/sqm, cap rates, buyer profiles

Medium

Risk Scoring

Systematic assessment across 5 dimensions (planning, tenant, market, execution, regulatory) → scores 1-5 per dimension → generates risk heatmap

Complex

Financial Modeling

Auto-populates underwriting template → runs IRR/EM calculations → generates sensitivity tables (rent, cap rate, cost, timing) → produces upside/base/downside scenarios

Complex

Deal Dashboard

Synthesizes all analysis into client-ready 48-hour dashboard: executive summary, financial analysis, risk assessment, comps, key considerations, DD priorities

PROSPECTOR

Investor Intelligence & Outreach

8-12 hrs/week

Mundane

Profile Enrichment

Medium

Warm Path Mapping

Complex
